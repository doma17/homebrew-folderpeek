# homebrew-folderpeek

Homebrew tap for [FolderPeek](https://github.com/doma17/FolderPeek), a macOS Quick Look extension for previewing folder, zip, and tar contents from Finder.

## Install

```sh
brew tap doma17/folderpeek
brew trust doma17/folderpeek
brew install --cask folderpeek
```

## Release source

FolderPeek releases are published at:

```text
https://github.com/doma17/FolderPeek/releases/latest
```

FolderPeek 0.3 is an early-tester direct-distribution build. Until Developer ID signing and notarization are complete, macOS Gatekeeper may require manual approval on first launch.
